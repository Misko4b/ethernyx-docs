# ⚙️ Settings

<figure><img src="../.gitbook/assets/image (1).png" alt="" width="563"><figcaption></figcaption></figure>

## Wallet Settings:

In wallet settings you can see:

* A list of all your wallets
* Wallet balances
* OpenSea profile info
* Select wallet as main (It will be used as a default parameter when creating tasks, as well as its nickname and avatar will be displayed in your Ethernyx profile)

### Bidding Settings:

Default parameters:

* **Default Marketplace**: Choose which marketplace will be used by default when creating tasks (values: Blur, OpenSea, All)
* **OpenSea Default Outbid Margin**: Select which margin multiplier will be used by default on OpenSea
* **OpenSea Increased Outbid Margin**: Select which increased margin multiplier will be used by default on OpenSea (only for bid modes: smart, smart\_ids, ids)
* **Blur Outbid Margin**: Select which margin will be used by default on Blur

BETH deposit and ETH Wrapping:

* **Deposit Destination**: Selecting which marketplace your funds will be sent to automatically (OpenSea, Blur, Separate (50/50))
* **Minimum Wallet Balance**: What balance should remain on your wallet after the deposit of funds to the marketplace (Select 9999 to disable the automatic deposits)

### Listing Settings

* **Default Marketplace**: Choose which marketplace will be used by default when creating tasks (values: Blur, OpenSea, All)
* **Listing Duration**: Select for how long the nft is put up for sale (minimum 15 minutes)
* **Max Gas Price (GWEI)**: Limit of gas price for on-chain transactions (BETH deposit, WETH wrap, approve collection for sale)

{% hint style="info" %}
After changing the settings, be sure to click on the “Save Changes” button
{% endhint %}
