# 🐣 First-Time Users

1. Connect your wallet on [ethernyx.io](https://app.ethernyx.io/).
2. Create a new wallet or import an existing one by following the instructions in [Wallet Setup](../wallet-setup.md)\
   &#xNAN;_&#x57;hen creating a new wallet, make sure to save the private keys generated and also consider importing the wallet to a hot wallet like MetaMask or Rabby!_
3. Fund your wallet by transferring funds to your wallet address.
4. You're now ready to start bidding!
